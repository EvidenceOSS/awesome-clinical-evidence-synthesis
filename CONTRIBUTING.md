# Contributing to awesome-clinical-evidence-synthesis

Thank you for your interest in contributing! We welcome contributions from the community.

## How to Contribute

1. **Open an Issue** — Report bugs, suggest new entries, or propose improvements via [GitHub Issues](https://github.com/EvidenceOSS/awesome-clinical-evidence-synthesis/issues).
2. **Fork & Branch** — Fork this repository and create a feature branch from `main`.
3. **Make Changes** — Follow the existing format and style conventions.
4. **Submit a Pull Request** — Open a PR against `main` with a clear description of your changes.

## Guidelines

- Ensure all links are valid and point to actively maintained resources.
- Follow the existing categorization structure.
- One resource per line, with a brief description.
- Keep descriptions concise and objective.

## Code of Conduct

Be respectful and constructive. We are committed to providing a welcoming and inclusive experience for everyone.

## Questions?

Open an issue or reach out via the repository discussions.
