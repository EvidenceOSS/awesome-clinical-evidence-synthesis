# awesome-clinical-evidence-synthesis

![Status: Reserved](https://img.shields.io/badge/Status-Reserved-lightgrey?style=flat-square)

> 100+ curated tools for systematic literature review, meta-analysis, and evidence synthesis.

## Overview

A curated list of software tools, frameworks, libraries, and resources for clinical evidence synthesis. Covers the full pipeline from search strategy and screening through data extraction, meta-analysis, GRADE assessment, and living review automation. Designed for researchers, methodologists, and health-tech teams building evidence-informed products.
